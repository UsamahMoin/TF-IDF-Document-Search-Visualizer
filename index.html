Weighting model

What ltc.lnc means.

SMART notation independently specifies term frequency, document frequency, and normalization for document and query vectors.

Document vector ltc
llogarithmic TF1 + log10(tf)
tlogarithmic IDFlog10(N / df)
ccosine normalizationweight / ||d||
Query vector lnc
llogarithmic TF1 + log10(tf)
nno IDF factorglobal weight = 1
ccosine normalizationweight / ||q||
Final similarity score(d, q) = d · q

Because both vectors are normalized, their dot product is cosine similarity. Matching terms contribute in proportion to both query and document weight.

Research foundation

Primary sources behind the model.

The implementation follows foundational work on vector-space retrieval, automatic term weighting, and suffix stripping.

Salton, Wong & Yang · 1975 A Vector Space Model for Automatic Indexing

Introduces vector-space document representation and similarity-based retrieval.

↗ Salton & Buckley · 1988 Term-Weighting Approaches in Automatic Text Retrieval

Defines and evaluates the SMART family of local, global, and normalization choices.

↗ Porter · 1980 An Algorithm for Suffix Stripping

Describes the English stemming procedure used to consolidate related word forms.
